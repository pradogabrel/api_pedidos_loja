# 🛒 Sistema de Clientes e Vendas

Projeto desenvolvido com foco em gerenciamento de clientes e registro de vendas utilizando ASP.NET, SQL Server e Frontend Web.

---

## 🚀 Tecnologias Utilizadas

### Backend

* C#
* ASP.NET Web API
* Entity Framework
* SQL Server

### Frontend

* HTML5
* CSS3
* JavaScript

---

## 📂 Estrutura do Projeto

```bash
Projeto-Loja/
│
├── ApiDBLojaJV/
│   ├── Controllers/
│   ├── Models/
│   ├── Data/
│   └── Program.cs
│
├── FrontendLoja/
│   ├── index.html
│   ├── style.css
│   └── app.js
│
└── README.md
```

---

## ⚙️ Funcionalidades

✅ Cadastro de clientes
✅ Atualização de clientes
✅ Exclusão de clientes
✅ Registro de vendas
✅ Integração com banco SQL Server
✅ API REST em ASP.NET

---

## 🖥️ Interface

### Tela de Clientes

* Listagem de clientes
* Cadastro rápido
* Atualização e exclusão

### Tela de Vendas

* Registro de vendas
* Integração com clientes cadastrados

## ▶️ Como Executar o Projeto

### 1️⃣ Clonar o repositório

```bash
[git clone URL_DO_REPOSITORIO
](https://github.com/pradogabrel/api_pedidos_loja.git)```

---

### 2️⃣ Backend

Entrar na pasta da API:

```bash
cd ApiDBLojaJV
```

Executar:

```bash
dotnet run
```

---

### 3️⃣ Frontend

Abrir o arquivo:

```bash
index.html
```

ou utilizar a extensão Live Server no VS Code.
